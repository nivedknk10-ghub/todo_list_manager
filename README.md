# To-Do List Manager

## Description
This is a simple command-line To-Do List Manager built using Python.  
It allows users to add tasks, view tasks, mark tasks as completed, and save tasks to a file so they persist between program runs.

This project was created to learn project structure, GitHub workflow, and basic software development practices.

---

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Save tasks to a file
- Load tasks automatically on startup

---

## Project structure
todo_list_project/
